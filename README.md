<a name="zh-tw"></a>
# 鈴聲製作器

一個簡單易用的 Chrome 擴展，用於製作手機鈴聲。支持裁剪音頻文件的特定片段並下載為 WAV 格式。

## 官方支持

- 官方網站：[木奇鈴聲](https://tw.muqils.com/)
- 隱私政策：[隱私政策說明](https://tw.muqils.com/privacypolicy.html)

木奇鈴聲是專業的手機鈴聲下載網站，提供最新最全的手機鈴聲、短信鈴聲等下載服務。本擴展是其官方提供的鈴聲製作工具。

## 功能特點

- 支持多種音頻格式 (MP3, WAV, OGG, M4A, AAC)
- 可視化音頻波形顯示
- 直觀的區域選擇功能
- 實時音頻預覽
- 支持多語言 (繁體中文, 簡體中文, 英文, 日文, 韓文)
- 最大支持 50MB 的音頻文件
- 導出為標準 WAV 格式

## 安装方法

1. 从 Chrome 网上应用店安装
   - 访问 [Chrome 网上应用店页面]()
   - 点击"添加到 Chrome"

2. 开发者模式安装
   - 下载源代码并解压
   - 打开 Chrome 扩展管理页面 (chrome://extensions/)
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择解压后的文件夹

## 使用方法

1. 点击 Chrome 工具栏中的扩展图标
2. 点击"选择本地音频文件"按钮
3. 在波形图上拖动选择要裁剪的区域
4. 使用播放按钮预览选中区域
5. 点击"裁剪并下载"保存铃声

## 技术栈

- WaveSurfer.js - 音频可视化
- Web Audio API - 音频处理
- Chrome Extension API - 国际化支持
