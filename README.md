# 鈴聲製作器

一個簡單易用的 Chrome 擴展，用於製作手機鈴聲。支持裁剪音頻文件的特定片段並下載為 WAV 格式。

## 官方支持

- 官方網站：[木奇鈴聲](https://tw.muqils.com/)
- 隱私政策：[隱私政策說明](https://tw.muqils.com/privacypolicy.html)

木奇鈴聲是專業的手機鈴聲下載網站，提供最新最全的手機鈴聲、短信鈴聲等下載服務。本擴展是其官方提供的鈴聲製作工具。

## 功能特點

- 支持多種音頻格式 (MP3, WAV, OGG, M4A, AAC)
- 可視化音頻波形顯示
- 直觀的區域選擇功能
- 實時音頻預覽
- 支持多語言 (繁體中文, 簡體中文, 英文, 日文, 韓文)
- 最大支持 50MB 的音頻文件
- 導出為標準 WAV 格式

## 安裝方法

1. 從 Chrome 網上應用店安裝
   - 訪問 [Chrome 網上應用店頁面]()
   - 點擊「添加到 Chrome」

2. 開發者模式安裝
   - 下載原始碼並解壓
   - 打開 Chrome 擴展管理頁面 (chrome://extensions/)
   - 開啟「開發者模式」
   - 點擊「載入未封裝項目」
   - 選擇解壓後的資料夾

## 使用方法

1. 點擊 Chrome 工具列中的擴展圖示
2. 點擊「選擇本地音頻文件」按鈕
3. 在波形圖上拖動選擇要裁剪的區域
4. 使用播放按鈕預覽選中區域
5. 點擊「裁剪並下載」儲存鈴聲

## 技術棧

- WaveSurfer.js - 音頻可視化
- Web Audio API - 音頻處理
- Chrome Extension API - 國際化支持

# Ringtone Maker

A simple and easy-to-use Chrome extension for making phone ringtones. Supports trimming specific segments of audio files and downloading them in WAV format.

## Official Support

- Official Website: [MuQi Ringtones](https://www.muqils.com/)
- Privacy Policy: [Privacy Policy Details](https://www.muqils.com/privacypolicy.html)

MuQi Ringtones is a professional ringtone download website, providing the latest and most comprehensive mobile ringtones and message tones for download. This extension is their official ringtone creation tool.

## Features

- Supports multiple audio formats (MP3, WAV, OGG, M4A, AAC)
- Visual audio waveform display
- Intuitive region selection
- Real-time audio preview
- Multi-language support (Traditional Chinese, Simplified Chinese, English, Japanese, Korean)
- Supports audio files up to 50MB
- Exports to standard WAV format

## Installation

1. Install from Chrome Web Store
   - Visit [Chrome Web Store Page]()
   - Click "Add to Chrome"

2. Developer Mode Installation
   - Download and extract the source code
   - Open Chrome extensions page (chrome://extensions/)
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the extracted folder

## How to Use

1. Click the extension icon in Chrome toolbar
2. Click "Select Audio File" button
3. Drag on the waveform to select the region to clip
4. Use play button to preview the selected region
5. Click "Clip & Download" to save the ringtone

- WaveSurfer.js - Audio visualization
- Web Audio API - Audio processing
- Chrome Extension API - Internationalization support

# 着信音メーカー

Chrome拡張機能で簡単に着信音を作成できます。音声ファイルの特定部分を切り取り、WAV形式でダウンロードすることができます。

## 公式サポート

- 公式サイト：[木奇着信音](https://www.muqils.com/)
- プライバシーポリシー：[プライバシーポリシーの詳細](https://www.muqils.com/privacypolicy.html)

木奇着信音は、最新かつ豊富な携帯電話の着信音やメッセージ音をダウンロードできる専門サイトです。この拡張機能は公式の着信音作成ツールです。

## 特徴

- 複数の音声フォーマットに対応 (MP3, WAV, OGG, M4A, AAC)
- 音声波形の可視化表示
- 直感的な範囲選択
- リアルタイムプレビュー
- 多言語対応 (繁体字中国語, 簡体字中国語, 英語, 日本語, 韓国語)
- 最大50MBまでの音声ファイルに対応
- 標準WAV形式でエクスポート

## インストール方法

1. Chrome ウェブストアからインストール
   - [Chrome ウェブストア]()にアクセス
   - 「Chrome に追加」をクリック

2. 開発者モードでインストール
   - ソースコードをダウンロードして解凍
   - Chrome の拡張機能ページ(chrome://extensions/)を開く
   - 「デベロッパーモード」を有効にする
   - 「パッケージ化されていない拡張機能を読み込む」をクリック
   - 解凍したフォルダを選択

## 使用方法

1. Chrome ツールバーの拡張機能アイコンをクリック
2. 「音声ファイルを選択」ボタンをクリック
3. 波形上でドラッグして切り取る範囲を選択
4. 再生ボタンで選択範囲をプレビュー
5. 「切り取ってダウンロード」をクリックして着信音を保存

## 技術スタック

- WaveSurfer.js - 音声の可視化
- Web Audio API - 音声処理
- Chrome Extension API - 多言語対応
